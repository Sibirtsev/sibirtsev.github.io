# Структура изображений

## Расположение изображений

### static/images/
Основная папка для изображений. Файлы отсюда доступны напрямую по пути `/images/...`

#### static/images/posts/
Обложки и изображения для постов блога.

Пример использования в посте:
```yaml
---
image: "/images/posts/my-post-cover.jpg"
---
```

#### static/images/portfolio/
Обложки и скриншоты для проектов портфолио.

Пример использования в проекте:
```yaml
---
image: "/images/portfolio/my-project.jpg"
---
```

## Рекомендации по изображениям

### Размеры
- **Обложки постов**: 1200x630px (соотношение 1.91:1)
- **Проекты портфолио**: 1200x800px (соотношение 3:2)
- **Аватар**: 400x400px (квадрат)

### Форматы
- **Фото**: `.jpg` или `.webp`
- **Иллюстрации**: `.png` или `.webp`
- **Иконки**: `.svg` (предпочтительно)

### Оптимизация
- Сжимайте изображения перед загрузкой (TinyPNG, Squoosh)
- Используйте WebP для лучшей производительности
- Максимальный размер файла: 200-300KB

## Альтернативный вариант: Page Bundles

Если хотите хранить изображения рядом с контентом:

```
content/ru/posts/my-post/
├── index.md        # Пост
└── cover.jpg       # Обложка
```

В frontmatter используйте:
```yaml
---
image: "cover.jpg"
---
```
